Nacero
Motion Design Concept

The below is an overview of the Motion Design Concept. Each of these animations is a video file, the implementation properties for each example can be found in the Nacero Design Library under motion.

Atoms

01 — Typography

Heading — speed 1

Headlines are animated per character. To ensure the load time is always pleasingly brief there are different speeds for different lengths of line. Speed 1 is the slower of the two. And should be used only for short statements and the largest the text sizes. 

Heading — speed 2

Headlines are animated per character. To ensure the load time is always pleasingly brief there are different speeds for different lengths of line. Speed 2 is the faster of the two, and should be used for longer statements and smaller text sizes. 

body

Body text is animated by line. There is only one speed for both bold and book weights.

body copy sideways

Body text can also react to lateral movements by its parent object. This is effect is animated by line.

block quote

Block quote animation is scaled up version of our sequence for body text. In addition to the block quote component this can be used when line lengths are too long for ‘heading-speed-2’.

number block

Our numbers count up from the last 5% of the value for larger numbers and the last 50% of the value for smaller numbers. Supporting type uses our standard ‘body’ animation and the numbers themselves use ‘block-quote’.

Molecules

02 — Micro interactions

links

Link underlines enter from the left and exit on the right. There is a delay on each line creating visual interest.

primary CTA

On hover a highlight slides in from the left. Our Primary CTA uses our brand highlight colour.

secondary CTA

On hover a highlight slides in from the left. The secondary CTAs use either white or black depending on the underlying content. 

Timer tabs

Each Tab’s line (timer) can be set to a preset time. For example 7 seconds. When activated Timer Tabs appear at 100% opacity. At any time the user can tap/click a tab and the sequence will begin again from that point. 

Sequences

03 — Appearance

global header

On load the Global Header loads sequentially from left to write mimicking the action of the headline typography.

Hero sections

Images load at 120% size and scale down creating the impression content lands on the page. Other elements then enter sequentially in a cascade. 

subject block

Type cascades in sequentially according to the effects defined in the typography section with the support icons appearing last.

standard sections

The image enters first with the same animation as hero sections but cropped within the image frame. Type then enters in a cascade in the following order: heading, body, and lastly icons and micro copy.

article cards

Each layer of the component loads slightly offset, this happens both inside the cards and with supporting elements.  

Organisms

04 — Component Interactions

Menu interaction

When the user hovers over a header link with an arrow a dropdown appears with a preview of each content type. If they click or hover over another header link the dropdown dissappears.

Mobile Menu interaction

When the user taps the hamburger icon the menu fades in and type cascades in. The menu expands and collapses where carats are present. The hamburger icon animates.

related links highlight (strong)

On hover; the green highlight layer appears from the left and stretches across the row. At same time it subtly affects the other elements including the text and the image. When the highlight passes beneath the icon the icon changes to white and vice versa.

resource links highlight (weak)

On hover grey highlight layer appears from the left and stretches across the row, When the highlight passes beneath the icon the icon changes to blue and vice versa.

Team member hover

On hover, the profile image is replaced by an image with the employee and their vehicle.

Testimonials

Testimonials change dynamically, with each quote given a couple of seconds before switching to the next. Photos of people can be tapped / clicked.

Subject Cards

Elements within the card inherit the movement of the card itself with each element exhibiting its own inertia.

Dynamic

05 — Scroll

Sticky Navigation

Sticky navigation appears dynamically whenever the user scrolls up the page. When the user reaches the top of the page or scrolls down the sticky navigation dissappears.

Parallax effect in scroll

Parallax creates an illusion of depth whereby ‘closer’ objects move more quickly and ‘further away’ objects move more slowly. The effect is driven relative to the scroll position.
